# User Authentication System

A Django-based authentication system where users can register, log in, and view their profiles. Administrators can view and manage all registered users.

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS

## Features

- User Registration
- User Login
- User Logout
- User Profile Management
- Admin Dashboard
- User Data Management
